# PushRun

러닝 대회 접수 시작 시간을 놓치지 않도록 알림을 설정하는 PushRun 실행 화면입니다.

## 실행

가장 쉬운 방법:

1. `outputs/serve-pushrun.ps1` 실행
2. 브라우저에서 `http://127.0.0.1:4173/` 열기

서버 없이 파일로 열어도 화면은 볼 수 있습니다.

```txt
outputs/pushrun-site/index.html
```

## 주요 기능

- 대회 목록/상세
- 검색/필터
- 정각, 20분 전, 10분 전 알림 예약 UX
- 내 알림 목록
- 알림 권한 및 배터리 최적화 안내
- 공식/러너블/마라톤GO에서 확인된 접수 페이지가 있을 때만 접수 확인 버튼 표시
- 접수일과 대회일을 카드 안에서 분리해 모바일에서도 빠르게 확인

웹 실행 화면의 예약 알림은 브라우저가 열려 있는 동안 동작합니다. 실제 iOS/Android 앱에서는 휴대폰 알림으로 예약됩니다.
