Ready to Run

알림이 살아있어야 접수 순간을 잡아요.

PushRun이 브라우저 알림으로 접수 시간을 알려드릴 수 있게 아래 두 가지를 확인해주세요.

알림 허용

20분 전, 10분 전, 정각 팝업을 받습니다.

배터리 제한 해제

휴대폰이 앱을 잠재워도 알림이 끊기지 않게 합니다.

현재 웹 버전은 브라우저가 알림을 보낼 수 있는 상태에서 가장 안정적입니다.

Battery

알림이 늦지 않게 배터리 제한을 풀어주세요.

Android는 설정 화면으로 이동할 수 있고, iPhone은 설정 앱에서 저전력 모드를 확인하면 됩니다.

Android

설정 → 앱 → PushRun → 배터리 → 제한 없음 또는 최적화 제외를 선택하세요.

iPhone

설정 → 배터리에서 저전력 모드를 꺼두면 알림과 백그라운드 동작이 더 안정적입니다.
