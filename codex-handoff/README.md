# 집에서 Codex 이어가기

회사 PC Codex 채팅 목록을 집에서 복구하기 위한 인계 폴더다.

## 집에서 Codex에 붙여넣을 문장

```text
아래 GitHub 파일을 열어서 회사 PC Codex 채팅 목록을 복구해줘.

https://github.com/runnerpyrri-lgtm/pushrun/blob/main/codex-handoff/codex-6-thread-restore-index.md

나는 회사 PC Codex 사이드바에 있던 채팅들을 집에서 이어가고 싶어.
각 채팅별로 어떤 대화였는지, 마지막 상태가 무엇인지, 이어서 하려면 뭘 말하면 되는지 정리해줘.

가능하면 먼저 "어떤 채팅을 이어갈까요?"라고 묻고, 내가 제목을 말하면 그 채팅의 맥락으로 바로 이어서 도와줘.
```

## 파일

- `codex-6-thread-restore-index.md`: 회사 PC 채팅별 상세 인계 내용.
- `codex-continuity-note.md`: 짧은 안내와 한계 설명.

## 주의

이 방식은 Codex 채팅창 자체를 동기화하는 것이 아니라, GitHub에 저장한 인계 문서를 읽고 맥락을 복구하는 방식이다.
