# 러닝봄 Google Play 비공개 테스트 안내

## 테스터가 확인할 흐름

1. 앱을 설치한 뒤 지역과 거리 필터를 선택한다.
2. `접수 예정` 상태이며 공식 접수 시각이 확인된 대회에서 `접수 알림 예약`을 누른다.
3. 공식 대회 페이지 버튼이 브라우저에서 열리는지 확인한다.
4. 네트워크를 끈 뒤에도 번들 대회 목록과 이미 예약한 기기 로컬 알림이 유지되는지 확인한다.
5. 오류, 잘못된 일정, 링크 문제는 `hello.robom@gmail.com`에 기기 모델·Android 버전·대회명·화면을 함께 보낸다.

## 개인정보과 데이터 보안

- 계정 생성, 광고 추적, 분석 SDK, 서버로의 개인 정보 전송은 사용하지 않는다.
- 대회 데이터와 알림 예약 정보는 기기와 운영 공개 JSON에서만 처리한다.
- 알림 권한은 사용자가 허용했을 때만 사용하고, 거부해도 대회 탐색과 공식 링크는 사용할 수 있다.
- 외부 대회·접수 사이트는 해당 운영자의 정책을 따른다.

## 심사 메모

러닝봄은 대회 주최자나 접수처를 대표하지 않는 독립 정보 도구다. 정확한 접수 조건과 최종 일정은 연결된 공식 페이지에서 다시 확인해야 한다.
