# 러닝봄

러닝 대회 접수 시작 시간을 놓치지 않도록 알림을 설정하는 러닝봄 실행 화면입니다.

## 실행

가장 쉬운 방법:

1. `outputs/serve-pushrun.ps1` 실행
2. 브라우저에서 `http://127.0.0.1:4173/` 열기

서버 없이 파일로 열어도 화면은 볼 수 있습니다.

```txt
outputs/pushrun-site/index.html
```

## 주요 기능

- 대회 목록/상세
- `races.json` 기반 대회 데이터
- 검색/필터
- 정각, 20분 전, 10분 전 알림 예약 UX
- 내 알림 목록
- 알림 권한 및 배터리 최적화 안내
- 공식/러너블/마라톤GO에서 확인된 페이지가 있을 때만 대회 페이지 버튼 활성화
- 접수일과 대회일을 카드 안에서 분리해 모바일에서도 빠르게 확인
- 접수일과 대회일까지 남은 날짜를 D-day로 표시
- 로봄 패밀리 5개 앱과 앱 메타데이터, 문의·개인정보 흐름
- 자체 PWA 설치 버튼과 iOS Safari 홈 화면 추가 안내
- 기본 비동의·무전송 분석 어댑터와 명시적 동의 설정

웹 알림의 브라우저 타이머는 RunningBom이 열려 실행 중일 때만 확인합니다. 서버 푸시 알림은 제공하지 않습니다.
