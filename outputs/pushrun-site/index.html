Ready to Run

알림이 살아있어야 접수 순간을 잡아요.

러닝봄이 브라우저 알림으로 접수 시간을 알려드릴 수 있게 아래 두 가지를 확인해주세요.

알림 허용

20분 전, 10분 전, 정각 팝업을 받습니다.

페이지 실행 유지

페이지를 닫거나 기기가 절전 상태가 되면 예약 알림이 울리지 않을 수 있습니다.

알림을 이미 "차단"했다면 브라우저 주소창 옆 사이트 설정(자물쇠 아이콘)에서 알림을 "허용"으로 바꿔야 다시 받을 수 있어요.

중요한 접수 일정은 대회 공식 사이트와 휴대폰 캘린더에도 함께 저장해 주세요.

Web Alert

현재 웹 알림의 동작 범위입니다.

러닝봄은 서버 푸시가 아니라 이 페이지의 타이머로 접수 시각을 확인합니다.

가능한 경우

페이지가 열려 있고 브라우저 알림 권한이 허용된 상태입니다.

보장하지 않는 경우

페이지 종료, 브라우저 강제 종료, 기기 절전 상태에서는 알림이 울리지 않을 수 있습니다.
